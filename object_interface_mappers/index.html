Server Side UI Helpers

Server side HTML rendering in Rails is easy and concise because Rails offers reflection on both the data schema and routes.

This reflection capability is the enabling commonality that powers many Rails helpers and gems including the very popular simple_form and link_to.

<%= simple_form_for @user do |f| %>
  <%= f.input :username %>
  <%= f.input :password %>
  <%= f.button :submit %>
<% end %>

<%= link_to @user%>

TCOMB Validation

tcomb-validation - Validation library based on type combinators

var SignInInfo = struct({
  username: Str,
  password: Str
});

// retrieves values from the UI
var formValues = {
  username: $('#username').val().trim() || null,
  password: $('#password').val().trim() || null
};

// if formValues = {username: null, password: 'password'}
var result = validate(formValues, SignInInfo);
result.isValid();     // => false
result.firstError();  // => 'username is `null`, should be a `Str`'
